# 🚦 Accident Trends & Safety Insights Dashboard

An interactive Power BI dashboard built to analyze road accident data and uncover meaningful patterns related to accident frequency, severity, and contributing factors.

---

## 📊 Overview

This project transforms raw accident data into clear, actionable insights using Power BI. It focuses on understanding how different factors such as road type, vehicle category, weather conditions, and location influence accident trends and severity.

---

## 🔑 Key Highlights

* Designed interactive dashboards with **dynamic KPIs** like total accidents, total casualties, and average casualties per accident
* Performed **time-based analysis** to track monthly trends and year-over-year changes
* Explored accident distribution across **vehicle types, road types, and light conditions**
* Compared **urban vs rural accident patterns** to identify high-risk areas
* Implemented **slicers and filters** for flexible and user-driven analysis

---

## 🧠 Insights

* Urban areas show a higher concentration of accidents compared to rural regions
* Accident severity tends to increase during low-light or night conditions
* Cars contribute to the majority of total casualties
* Single carriageway roads account for the highest number of incidents

---

## 🛠️ Tech Stack

* **Power BI** – Dashboard development & visualization
* **DAX** – Calculated measures and KPIs
* **Power Query** – Data cleaning and transformation

---

## 📁 Files Included

* Power BI Dashboard file (`.pbix`)
* Dataset used for analysis (`.csv`)

---

## 🎯 Objective

To present accident data in a structured and visual format that helps in identifying trends, understanding risk factors, and supporting better decision-making for road safety.

---
